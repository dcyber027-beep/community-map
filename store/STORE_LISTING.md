## App Name

Melbourne Community Map

## Short Name

Community Map

## Short Description

Community safety awareness for Melbourne: see reports, post updates, and share short-lived street notes.

## Full Description

Melbourne Community Map helps people stay aware of what is happening nearby. View community-submitted reports on a live map, filter recent activity, add short-lived street notes, and join an anonymous community chat.

The app is designed for awareness, not emergency response. If someone is in immediate danger in Australia, call 000.

Key features:

- Interactive Melbourne map with recent community reports
- Filters for urgency and recent time windows
- Optional GPS, address search, or pin-drop reporting
- Short-lived street notes for local context
- Anonymous community chat that clears automatically
- Admin-managed street highlights and public notices
- Installable app experience with offline shell support
